#### KTH LARGEST MEANS ,GO FROM LARGEST LIST
[1,2,3,4,5]

5,4,3,2,1
4 is 2nd largest



```python
import heapq
class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        if len(nums)==1:
            return nums[0]
        for i in range(len(nums)):
            nums[i]=-nums[i]
        heapq.heapify(nums)
        x=heapq.nsmallest(k,nums)
    
        return -x[-1]

```
O(nlogk)
